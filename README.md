# Braille Typing Training Device

## Overview
This project is an assistive Braille typing training device designed to help users practice Braille input through physical buttons and audio feedback.

## Features
- Braille dot input using push buttons
- Character recognition and text conversion
- Audio feedback for user interaction
- Raspberry Pi Pico 2-based embedded system
- Custom PCB design using KiCad

## Technologies Used
- Raspberry Pi Pico 2
- C/C++
- KiCad
- Embedded Systems
- PCB Design
- Electronics Testing

## Project Status
Currently under development as part of an engineering internship at KOSEN, Japan.
